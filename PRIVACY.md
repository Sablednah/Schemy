# Privacy policy

Schemy is a local desktop viewer for Minecraft structure files.

## Data collection

Schemy does not collect analytics, telemetry, personal information, usage data, or crash reports. It does not contain advertising or user tracking.

## Files and network access

Structure files opened in Schemy are read and rendered locally. Their contents and rendered previews are not uploaded or transmitted to the project maintainers or any third party.

The Windows Explorer Preview Handler communicates with the local Schemy preview broker through a named pipe on the same computer. This communication does not leave the device.

This program will not transfer any information to other networked systems unless specifically requested by the user or the person installing or operating it.

Links opened from the project documentation, GitHub release pages, and third-party distribution services are governed by the privacy policies of those services. Code-signing services process build artifacts during the release process; they do not receive files viewed by Schemy users.

## Changes and contact

Material changes to this policy will be published in the public repository. Questions or concerns can be submitted through the project's [issue tracker](https://github.com/Sablednah/Schemy/issues).
